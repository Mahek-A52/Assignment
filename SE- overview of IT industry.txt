[5] What are the key steps involved in the programming process?
===>
     1. Understand the Problem 

First, clearly understand what you need to do
 Know:

What is the input?

What output is required?

2. Plan the Solution (Algorithm) 

Think step-by-step how to solve the problem
Write simple steps in order

3.Write the Program (Coding) 

 Convert your steps into a programming language (C, Python, etc.)

4. Run the Program 

Execute the program to see if it works

5. Test and Debug 

Check for errors (bugs)
Fix mistakes until correct output comes

6. Maintain and Improve 

Update the program if needed
